**Щоб запустити проєкт `MapTracker`, потрібно встановити залежності для клієнтської частини та `mock-server`:**

```bash
cd MapTracker
npm install

cd mock-server
npm install

cd ..
```

**Для запуску проєкту відкрийте дві вкладки термінала в кореневій директорії проєкту:**

```bash
npm run mock
```

— запуск mock-сервера

```bash
npm run preview
```

— запуск зібраної React-частини http://localhost:4173/
— доступ до карти за ключем demo-key-123
